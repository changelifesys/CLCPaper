開發說明

. 呼叫 Api\GetSUBInfo.aspx 取得課程資訊, 內容請讀取 HtmlSubDesc 欄位, 標題請讀取 SubName 欄位。(查看 API GetSUBInfo JSON 文件)
. 呼叫 Api\GetGroupItem.aspx 取得組別、小組的下拉選項。(查看 API GetGroupItem JSON 文件)
. 標註「*」的部分為必填欄位。
. 若勾選記住, 則使用者就不用再重填重覆欄位的資料
. 按下報名後，呼叫 Api\ChkMember.aspx 判斷是否有更改資料。(查看 API ChkMember JSON 文件)
